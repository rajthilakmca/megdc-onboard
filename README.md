megdcUI
=================

Automatted base install


### Requirements

>
[Golang 1.5 > +](http://www.golang.org/dl)


## Usage

``megdcui -v start``


### Compile from source


```
mkdir -p code/megam/go/src/github.com/megamsys

cd code/megam/go/src/github.com/megamsys

git clone https://github.com/megamsys/megdcui.git

cd megdcui

make

```
